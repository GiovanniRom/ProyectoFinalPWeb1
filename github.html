Github como herramienta

Github es una de las principales herramientas para desarrolladores en la actualidad ya que permite un manejo y control de versiones de forma eficiente, ademas de tner una comunidad muy activa que permite a los usuarios interactuar entre los diversos repositorios que se han publicado y obtener ideas o compartir las suyas

Caracteristicas de Github

Multiplataforma
Controlador de versiones
Gestor de Cambios
Codigo Abierto
Gratuito
Revision de Codigo
Red Social
Trabajo por ramas

Descripciones

Permite trabjar desde cualquier SO
A traves de su sistema Git permite llevar un registro sobre las versiones implementadas a lo largo de la historia del proyecto
Permite implementar unicamente aquellas actualizaciones que hayan sido previamente aprovadas
Los proyectos alojados en la platforma son de libre acceso para cualqueira que desee revisarlos
Los proyectos se almacenan en los servidores de Github de manera gratuita
Permite que el creado o demas usuarios revisen y comenten el codigo con posibles implementaciones de mejoras
Permite seguir a otro creadores de contenido para no perderte sus actualizaciones de proyectos
Permite trabajar en multiples ramas paralelas a la principal y converger los cambios cuando sea necesario
